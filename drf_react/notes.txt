* complete django installation is in the 'todo/backend/' subfolder
* Find the pip install file at 'todo/backend/requirements.txt'
* frontend installations
 - npm install -g create-react-app
* To create the frontend folder use the below commands
 - >create-react-app frontend
 - >cd frontend
 - >npm start
 - Check the react host -  http://localhost:3000/
 - write App.js

Note: Make sure you keep the django server running when you run the react server
